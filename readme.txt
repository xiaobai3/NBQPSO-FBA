# MATLAB NBQPSO-FBA

## Description
This MATLAB script optimizes metabolite production with Neighbourhood-based Quantum-Behaved Binary Particle Swarm Optimization and Flux Balance Analysis (EBQPSO-FBA)

## Installation
1. Make sure you have MATLAB installed on your machine.
2. Clone this repository or download the `NBQPSO-FBA.zip` file.
3. Unizp the `NBQPSO-FBA.zip` file as "NBQPSO-FBA" directory.
4. Make sure you have downloaded The COnstraint-Based Reconstruction and Analysis Toolbox.

##Usage
To optimize metabolite production, follow these steps:
1. Open MATLAB and navigate to the directory where you saved the "EBQPSO-FBA" directory.
2. In the MATLAB command window, type the following command:
   ```matlab
   mainForNBQPSO;
